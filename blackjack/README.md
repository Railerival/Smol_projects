## How to Play Blackjack?

Blackjack is a card game played between a **player** and a **dealer**.

### Objective

The goal of the game is to have a hand value that is:

* Higher than the dealer's hand value, **without exceeding 21**, or
* Exactly **21**, which results in an immediate win.

### Starting the Game

At the beginning of the game:

* The **player** receives **2 cards**.
* The **dealer** receives **2 cards**.

### Card Values

* Number cards (**2–10**) are worth their face value.
* **J**, **Q**, and **K** are each worth **10** points.
* **A (Ace)** can be worth either **1** or **11**, depending on which value gives the best possible hand.

### Gameplay

During the game, the player can choose one of the following actions:

#### Hit

* Take an additional card and add it to the current hand.

#### Stand

* Stop taking cards and keep the current hand value.

### Winning and Losing

* If a player's hand value exceeds **21**, the player **loses**.
* If the dealer's hand value exceeds **21**, the dealer **loses**.
* If the player reaches **21**, the player **wins immediately**.
* If neither side exceeds **21**, the winner is the one with the higher hand value.
* If the player and dealer have the same hand value, the game ends in a **draw**.
